# Linker
Linker is a url shortener built with Next.js

## Used technologies
- Typescript
- NextAuth
- tRPC
- Drizzle ORM
- TailwindCSS

### How to run the server
- Clone the repository
- Create a .env file following .env.example
- Run:
  ```bash
  npm run dev
  # or
  yarn dev
  # or
  pnpm dev
  ```
